# Resume Builder — 需求文档

## 项目目标
一个 AI 驱动的简历构建平台，支持用户创建、编辑、导出简历，使用模板市场，并通过 AI 解析和生成简历内容。

## 用户角色
- 普通用户：注册/登录后管理自己的简历和模板

## 功能需求

### 1. 用户认证
- 用户注册（用户名 + 密码）
- 用户登录，返回 JWT token
- 前端路由守卫，未登录跳转 `/login`

### 2. 简历管理
- 创建简历（选择模板或空白）
- 编辑简历内容（基本信息、教育经历、工作经历、技能、项目等）
- 查看简历列表
- 删除简历

### 3. 模板系统
- 模板市场（浏览公开模板）
- 创建/编辑自定义模板（HTML + CSS）
- 模板预览
- 模板应用到简历

### 4. AI 功能
- 上传 PDF/Word 简历文件，AI 解析提取结构化数据
- AI 生成/优化简历内容
- 解析任务异步处理，支持查看任务状态列表

### 5. 简历导出
- 导出为 Word (.docx)
- 基于模板渲染 HTML

## 非功能需求
- 后端 API 统一返回格式 `ApiResponse<T>`
- 数据库迁移通过 Flyway 管理
- AI 调用通过 Spring AI 封装，模型可配置
- 前端使用 TDesign 组件库保持 UI 一致性
