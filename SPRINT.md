# Resume Builder 迭代计划

## 项目概述

AI 驱动的简历构建系统，支持多模板管理、智能内容生成、简历编辑和导出。

---

## 技术栈

### 后端
- Spring Boot 3.2.0
- Spring Data JPA
- PostgreSQL 17
- JDK 17
- SpringDoc OpenAPI (Swagger)

### 前端
- Vue 3.4
- Vite
- TDesign UI 组件库
- Pinia (状态管理)
- Vue Router 4
- Axios

## 注意事项

### 开发原则
1. **分步实现**：每个 Sprint 完成后再开始下一个
2. **小步快跑**：每个功能模块独立开发和测试
3. **持续集成**：代码完成后立即测试和集成
4. **文档同步**：代码变更同步更新文档

### 技术债务
- [ ] JWT Token 简化版，需要升级为完整版
- [ ] 密码明文存储，需要使用 BCrypt 加密
- [ ] 缺少完整的异常处理
- [ ] 缺少日志记录

### 已知问题
- npm 依赖安装问题（前端）
- 登录状态持久化未完成
- 前端路由守卫未实现
